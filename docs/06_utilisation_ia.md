# Utilisation de l'IA

Cours : Veille technologique (420-1SH-SW)  
Sujet : Serveur IA self-hosted avec Ollama pour le support technique

---

## Déclaration d'utilisation

Dans ce projet, j'ai utilisé des outils d'intelligence artificielle comme aide à l'organisation, à la recherche d'idées et à la reformulation. Cependant, les LLM comme ChatGPT, Claude, Gemini ou Copilot ne sont pas utilisés comme sources primaires ou secondaires dans le rapport. Les informations importantes sont vérifiées à partir de sources officielles, institutionnelles ou spécialisées.

---

## Ce que l'IA a été utilisée pour faire

| Utilisation | Outil | Autorisé selon les consignes du cours |
|---|---|---|
| Brainstorming pour choisir le sujet du projet | Claude | Oui |
| Organisation du plan de recherche et du plan du rapport | Claude | Oui |
| Clarification de concepts techniques (comment fonctionne une API REST, qu'est-ce qu'un system prompt) | Claude | Oui |
| Reformulation de passages pour les rendre plus clairs | Claude | Oui |
| Correction de la langue française dans les documents | Claude | Oui |
| Aide à la structure des fichiers Markdown pour Obsidian | Claude | Oui |

---

## Ce que l'IA n'a PAS été utilisée pour faire

| Utilisation interdite | Raison |
|---|---|
| Source primaire dans le rapport | Les LLM peuvent halluciner des faits, inventer des chiffres et des références |
| Source secondaire dans le rapport | Les LLM ne sont pas révisés par des pairs et n'ont pas de processus de vérification factuelle |
| Remplacement de la recherche dans des sources officielles | Les faits du rapport sont tirés de sources vérifiables (NIST, OWASP, documentation officielle Ollama, etc.) |
| Génération de chiffres ou de statistiques présentés comme vrais | Tout chiffre cité dans le rapport provient d'une source vérifiable et citée |

---

## Limites des LLM à garder en tête

- Les LLM peuvent inventer des informations qui semblent vraies (hallucinations)
- Ils ont une date de coupure de connaissance et peuvent ignorer des développements récents
- Ils peuvent contenir des biais selon leurs données d'entraînement
- Ils ne donnent pas toujours des sources vérifiables
- Ils ne sont pas soumis à un processus de révision par les pairs
